# SLAM 기반 자율주행 로봇 시스템

컴퓨터 비전과 자율주행을 활용한 멀티 로봇 시스템 개발 프로젝트입니다.

---

## 프로젝트 개요

물류 자동화 및 스마트 팩토리 환경에서 다수의 로봇이 협력하는 멀티 로봇 시스템을 구현합니다.

- **기간:** 총 10일 (2주)
- **인원:** 3~5인 팀 프로젝트
  - 1~4일차: 4인 1조 개별 프로젝트
  - 5~10일차: 두 팀 통합 대규모 프로젝트

---

## 기술 스택

| 구분 | 내용 |
|------|------|
| OS | Ubuntu 22.04 |
| Framework | ROS2 Humble, Nav2, SLAM, AMCL, Costmap, YOLO, PyQt |

| H/W | TurtleBot4 |
| Simulator | Gazebo |
| 협업 도구 | Git, GitHub, Slack, Notion |

---

## 시작하기

### 환경 요구사항

- Ubuntu 22.04
- ROS2 Humble
- Python 3.10+

### 설치

```bash
# 저장소 클론
git clone <repository-url>
cd <repository-name>

# 의존성 설치
pip install -r requirements.txt

# ROS2 패키지 빌드
colcon build
source install/setup.bash
```

---

## 주요 기능

- **비전 인지:** YOLO 기반 객체 인식 + Point Cloud 3D 공간 분석
- **자율 주행:** SLAM 지도 생성, 경로 계획, Nav2 Navigation 튜닝
- **로봇 협업:** 다중 로봇 상태 공유 및 협력 미션 자동 수행
- **관제 시스템:** 실시간 모니터링 및 원격 제어 GUI
